# Simulador-de-investimento-excel
Ao iniciar o bootcamp Santander - Excel com Inteligência Artificial, um dos primeiros projetos foi criar um simulador de investimentos para facilitar o cálculo para pessoas que não sabem exatamente quanto, como e qual tipo de Fundo de investimento imobiliário deve investir. 

Esse simulador foi construído para nortear pessoas que precisam de um norte no que se refere a aplicação em FII (Fundo de investimento imobiliário). Ao inserir as informações, o simulador vai trazer as seguintes informações para o futuro investidor:
1. Valor de investimento mínimo recomendado, de acordo com a renda do indivíduo;
2. Valor de rentabilidade mensal e dividendos;
3. Recomendação de percentuais para cada tipo de fundo, conforme perfil do investidor sinalizado;
4. Cenário futuro do investimento de acordo com o prazo que determinar investir, alinhando melhor as expectativas de retorno.

Esse projeto fez parte de uma primeira atividade que fiz no Bootcamp Santander - Excel com Inteligência Artificial.
